Um pouco sobre HTML

HTML (HyperText Markup Language) é um padrão de marcação para páginas e aplicações web, e surgiu no início dos anos 90.

Atualmente a linguagem está na versão 5, popularmente conhecida como HTML5.

fonte: HTML - Wikipedia, the free encyclopedia

A estrutura de uma página HTML

Em sua composição, toda página HTML tem um padrão mínimo de elementos, chamados também de 'tags'. Abaixo a estrutura mínima de uma página HTML.

	<html>
		<head>
		<⁄head>				
		<body>
		<⁄body>
	<⁄html>

Neste capítulo inicial falaremos das tags <p>, <div>, <h1>, <b>, <em>, <span>.

Antes de falarmos sobre cada uma, é importante sabermos que as tags, com pequenas exceções, e trataremos essas, tem o mesmo padrão de formato, onde temos 'atributos' e seus respectivos 'valores' para cada tag. Vejamos abaixo:

    <TAG ATRIBUTO1="VALOR" ATRIBUTO2="VALOR" ...>Texto de conteúdo<⁄TAG>

Tag h1

A tag "h1" é utilizada para destacar, principalmente, títulos em um texto.

Sua estrutura é a seguinte:

	<h1>Título principal da página<⁄h1>

Exemplo:

	<h1>
Meu Título Principal
<⁄h1>

Obs.: Esta tag também pode ser utilizada em imagens, seguindo a mesma estrutura.


Tag p

A tag "p" é uma tag de bloco de conteúdo. Muito usado para estruturas de parágrafos em textos.

Sua estrutura é a seguinte:

	<p>Linha de texto de um texto principal.<⁄p>

Exemplo:

	<p>
Este é um parágrafo de texto. Você pode inserir o seu texto aqui.
<⁄p>

Tag br

Se você reparar, a tag 'p' automaticamente força o texto a ir para a próxima linha. E se eu precisar de mais uma linha pulada? Aí podemos usar a tag br.

Sua estrutura é a seguinte:

	<br>	-- (Pulando uma linha)

Exemplo:

	<br> -- (Pulando uma linha)
	Texto
	<br><br>
	

	(Pulando duas linhas)

Tag b

A tag "b" é uma tag usada para transformar o texto em negrito.

Sua estrutura é bem simples:

	<b>Texto em negrito<⁄b>

Exemplo:

	<b>Este é um texto escrito com a tag 'b'.<⁄b>

Tag em

A tag "em" é uma tag usada para transformar o texto em itálico. Antes a tag 'i' era utilizada, mas ela foi descontinuada e substituída.

Sua estrutura também é muito simples:

	<em>Texto em itálico<⁄em>

Exemplo:

	<em>Este é um texto escrito com a tag 'em'.<⁄em>

Tag span

A tag "span" é uma tag utilizada para textos, porém, com menor importância. Muito usada para colocar trechos de texto que não precisam de destaque ou importância, e que não necessitem de uma tag 'p'.

Sua estrutura é a seguinte:

	<span>Texto escrito com span<⁄span>

Exemplo:

	<span>Este é um texto escrito com span.<⁄span>

Tag a

A tag "a" é uma tag âncora, ou seja, ela é um hyperlink -- marcações ou pontos da página atual ou externa.

A propriedade href indica o link a direcionar. A estrutura é simples:

			<a href="link-a-se-acessado">Texto<⁄a>

Exemplo:

			Este é um de acesso à página da Udacity. Clique <a>aqui.<⁄a>

Tag img

A tag "image" é uma tag para marcação de hyperlinks, porém, imagens. Sua estrutura é um pouco diferente dos demais itens, pois tem a obrigatoriedade de duas propriedades.

A propriedade src indica o endereço da imagem, e a propriedade alt indica um texto alternativo caso a imagem não apareça.

Esta tag tem de diferente que também não possui uma tag de fechamento, como '<⁄p>'. Veja abaixo:

			<img src="https://br.udacity.com/assets/images/logo/logomark.min.svg" alt="texto-alternativo-a-imagem">
			 alt="text">

Exemplo:

			<img >

Tag div

A tag "div" é uma tag de bloco de conteúdo também, como a tag 'p'.

A diferença é que a tag div é comumente usada como um contêiner para organizar e estruturar os blocos de uma página HTML.

Sua estrutura é a seguinte:

	<div>
		<p>Primeira linha de texto.<⁄p>
		<img src="https://www.uol.com.br/img.svg" "texto-alternativo-a-imagem">
		<p>Segunda linha de texto.<⁄p>
	<⁄div>

Exemplo:

	<div>

		
Primeira linha de texto dentro da div. Link para a Udacity aqui.

		
Segunda linha de texto dentro da div. Link para a Udacity aqui.

		

	<⁄div>

Dicas Finais

Com base nos elementos apresentados, se você olhar o código fonte dessa página, verá que ela foi 99% escrita com os elementos apresentados -- com exceção de um único elemento chamado 'pre' que, resumidamente, formata um texto e o exibe da maneira que está no código. Podemos concluir o seguinte:

As tagas a, span, br, img, and strong são todos elementos 'inline', ou seja, elementos que não possuem blocos de outras tags neles. Isso não quer dizer que não possam ter, ok?!

Já os elementos div e p são blocos de elementos, ou seja, eles podem conter diversos elementos 'inline' neles.
