HTML

HTML (abreviação para a expressão inglesa HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto) é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. A tecnologia é fruto da junção entre os padrões HyTime e SGML.

Veja as versões já publicadas:

Versão Ano
HTML 1991
HTML 2.0 1995
HTML 3.2 1997
HTML 4.01 1999
XHTML 2000
HTML5 2014

Estrutura básica de um documento

Todo documento HTML tem uma estrutura básica, e aqui representaremos um desses modelos.

<!DOCTYPE html>
<html>
  <head>
    <charset="utf-8" />
    <meta name="description" content="a descrição do seu site em no máximo 90 caracteres">
    <meta name="keywords" content="escreva palavras-chaves curtas, máximo 150 caracteres">
    <title>Título do Documento<⁄title>
  <⁄head>
  <body>
  <!-- Dentro das tags <body> <⁄body> fica quase toda parte visível para todos, onde pode-se inserir
    textos, imagens, links para outras páginas, etc...
  geralmente usa-se: -->
    <div>Tag para criar-se uma 'caixa', um bloco, mais utilizada com "Cascading Style Sheets
        (Folhas de Estilo em Cascata)<⁄div>
    <span>Tag para modificação de uma parte do texto da página<⁄span>
    <img src="endereço_de_uma_imagem.jpg" ⁄>
    <a href="http://www.wikipedia.org">Wikipedia, A Enciclopédia Livre<⁄a>
  <⁄body>
<⁄html>

Dicas Finais

Com base nos elementos apresentados, se você olhar o código fonte dessa página, verá que ela foi 99% escrita com os elementos apresentados -- com exceção de um único elemento chamado 'pre' que, resumidamente, formata um texto, respeitando espaços definidos no arquivo HTML, e o elemento 'code', que geralmente é usado para exibir trechos de códigos em páginas HTML. Podemos concluir o seguinte:

As tagas a, span, br, img, and strong são todos elementos 'inline', ou seja, elementos que não possuem blocos de outras tags neles. Isso não quer dizer que não possam ter, ok?!

Já os elementos div e p são blocos de elementos, ou seja, eles podem conter diversos elementos 'inline' neles.
